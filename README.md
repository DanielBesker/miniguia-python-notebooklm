  <b>Projeto prático utilizando IA (NotebookLM) para aprendizado estruturado em Python</b><br>

---

🚀 Sobre o Projeto

Este repositório apresenta um caderno temático completo de Python, desenvolvido com apoio de Inteligência Artificial.

O objetivo é demonstrar:

- Capacidade de aprendizado estruturado
- Uso estratégico de IA
- Organização de conhecimento técnico
- Evolução do raciocínio (prompt engineering)

---

🎯 Objetivos

- Aprender Python do básico ao avançado
- Desenvolver lógica de programação
- Criar base para backend, automação e dados
- Praticar com exercícios reais
- Construir portfólio profissional
